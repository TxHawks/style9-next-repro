# style9-nextjs-example

Example using [style9](https://github.com/johanholmerin/style9) with Next.js.
See [next.config.js](next.config.js) for config and
[pages/index.tsx](pages/index.tsx) for usage.

## Installation

```sh
yarn
```

## Development

```sh
yarn dev
```

## Build

```sh
yarn build
yarn start
```
